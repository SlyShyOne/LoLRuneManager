It's recommended to pin the app to the taskbar or the start menu

*Import Usage:*
- Open a rune page in the client, and switch to grid view
- Then either:
  * Copy a compact rune page like this one (Just keep it in the clipboard):  CAN'T MILK THOSE::4,2,3,1,2,2,0,1,2
  * Or copy a probuilds/op.gg/etc page (see https://imgur.com/a/IpeHz)
- Activate the import program

*Export Usage:*
! Exporting cannot be done in champselect right now.
- Open a rune page in collections and activate the export program.
